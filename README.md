# Study

Учебный проект: банковское приложение на Python и PyQt6.

## Описание

Простое приложение для управления банковскими счетами с графическим интерфейсом. Позволяет создавать счета, пополнять и снимать средства, просматривать информацию по владельцам.

## Структура

```
study/
├── main.py          # Точка входа
├── gui/
│   └── app.py       # PyQt6 интерфейс
├── utils/
│   └── bank.py      # Класс Bank
└── requirements.txt
```

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Возможности

- Создание счёта (владелец + начальный баланс)
- Пополнение и снятие средств
- Просмотр всех счетов владельца
